# RSS Feed Aggregator

# Installation
Requirements: `go 1.22.5 or later`

- Clone this repo
- Run `go build .`
- Run `./rss-aggregator`
- Use the commands below

# Commands
```
- login <username>
- register <username>
- reset
- users
- agg
- feeds
- addfeed <name> <url>
- follow <url>
- unfollow <url>
- following
- browse <length?>
```
